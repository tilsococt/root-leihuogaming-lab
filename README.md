# root-leihuogaming-lab

这是一个用于归档和发布多个独立 HTML 页面的仓库。  
本仓库不针对任何特定网站或域名，仅作为页面资源的集中存放与展示用途。

---

## 项目简介

本仓库主要用于存放各类独立的 HTML 页面，内容涵盖实验性页面、存档页面、工具页面等。  
所有页面均为静态资源，不涉及后端服务或动态数据交互。

---

## 目录说明

```
.
├── index.html          # 仓库首页 / 入口页面
├── pages/              # 独立 HTML 页面存放目录
│   ├── example1.html
│   ├── example2.html
│   └── ...
├── assets/             # 页面共用资源（CSS、JS、图片等）
│   ├── css/
│   ├── js/
│   └── images/
└── README.md           # 本文件
```

- `pages/`：存放各独立页面文件，每个页面为一个完整的 HTML 文档。
- `assets/`：存放页面所需的公共静态资源，按类型分类。
